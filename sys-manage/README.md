## sys-manage

```lua
sys-manage -- 后台基础服务
│  │─sys-monitor -- 监控服务
│  │  ├─sys-monitor-api -- api
│  │  ├─sys-monitor-client -- client
│  │  ├─sys-monitor-core -- core
│  ├─sys-attach -- 附件管理服务
│  │  ├─sys-attach-api -- api
│  │  ├─sys-attach-client -- client
│  │  ├─sys-attach-core -- core
│  │─sys-org -- 组织架构服务
│  │  ├─sys-org -api -- api
│  │  ├─sys-org -client -- client
│  │  ├─sys-org -core -- core
```
